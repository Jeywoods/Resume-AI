## 📋 О проекте

AI Resume Builder — это веб-приложение, которое использует искусственный интеллект для создания профессиональных резюме. Просто опишите свой опыт, навыки и карьерные цели, а нейросеть сгенерирует структурированное резюме с современным дизайном.

### ✨ Особенности

| Функция | Описание |
|---------|----------|
| 🤖 **AI-генерация** | Описание вашего опыта → готовое резюме |
| ✏️ **Редактирование** | Все поля можно редактировать после генерации |
| 📄 **Экспорт в PDF** | Скачивание в формате PDF с сохранением дизайна |
| 📸 **Загрузка фото** | Добавление фотографии в резюме |
| 🚗 **Водительские права** | Указание категорий прав |
| 🎨 **Современный дизайн** | Чистый, профессиональный внешний вид |
